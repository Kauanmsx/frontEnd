# 🚀 TechSolutions — Landing Page

Landing page institucional desenvolvida como atividade prática da disciplina de Desenvolvimento Web, simulando o site de uma empresa de tecnologia voltada para soluções empresariais.

## 📖 Sobre o Projeto

A **TechSolutions** é uma startup fictícia especializada em serviços tecnológicos para pequenas e médias empresas. O projeto foi criado para praticar conceitos de desenvolvimento Front-End, focando em estruturação, estilização e interatividade.

## ✨ Funcionalidades

* Apresentação institucional da empresa
* Seção de serviços oferecidos
* Exibição da equipe
* Área de contato
* Integração com canal do YouTube
* Layout simples e responsivo

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## 📂 Estrutura do Projeto

```bash
TechSolutions/
│
├── img/
├── code.css
├── code.js
└── index.html
```

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/Kauanmsx/AtividadeCss.git
```

2. Acesse a pasta do projeto.

3. Abra o arquivo `index.html` em qualquer navegador.

## 🎯 Objetivos de Aprendizagem

* Estruturar páginas utilizando HTML5 semântico;
* Aplicar estilos com CSS3;
* Implementar interações básicas com JavaScript;
* Desenvolver uma landing page organizada e funcional;
* Praticar boas práticas de desenvolvimento Front-End.
